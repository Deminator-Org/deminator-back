# Deminator - BackEnd

This project is the back end part of a multiplayer minesweeper web-app.

# Dependencies

Java 1.16  
(JDK 1.16 for development)

# Install the projet

Unix: `./mvnw install`

Windows: `mvnw.cmd install`

# Run the project

Unix: `./mvnw spring-boot:run`

Windows: `mvnw.cmd spring-boot:run`
