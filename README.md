# Deminator - BackEnd

This project is the back end part of a multiplayer minesweeper web-app.

# Dependencies

Java 1.17
(JDK 1.17 for development)

# Install the projet

Unix: `./mvnw install`

Windows: `mvnw.cmd install`

# Run the project

Unix: `./mvnw spring-boot:run`

Windows: `mvnw.cmd spring-boot:run`

# Test the project

Unix & Windows:

```sh
cd test
./scenario#.sh
```
